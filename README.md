# GBAvatarBrowser
##仿写微信查看头像功能支持手势放大等<br>

#简介：<br>
##基本的头像查看类功能完全仿照微信，封装的是一个单例类<br>
#使用说明：<br>
* 使用方法十分简单将文件包拖入到你的项目中去<br>
* 创建一个单例直接用这个单例对象调用方法<br>
* 实例代码如下：<br>

```
 _myShower=[GBAvatarBrowser shareInstance];
 [_myShower showImage:iconBtn];
```
